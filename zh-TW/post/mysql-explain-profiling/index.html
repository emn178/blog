有時候在開發程式會遇到一些效能上的瓶頸，可能是 SQL 語法下的不好或者是索引建立不當所造成，不過在初期開發資料量不大的時候，可能無法明顯的感覺到效能上的差異，我們可以透過 EXPLAIN 語句來分析查詢語句的效能，確保在開發初期就能使用有效率的 SQL 語句。EXPLAIN 用法如下
