Linea 空投資訊整理

發表於2023-05-21|更新於2023-08-04|區塊鏈
|閱讀量:

Linea 是其中一個 ZK Rollup 的 L2 方案，背後公司是 Consensys，也是知名的錢包 Metamask 的開發公司。目前已經上主網，這邊研究並整理一份資料給大家參考。作為同樣 L2 ZK Rollup 的方案，策略就一樣參考之前的 zkSync 空投策略。

錢包

使用 Metamask 即可。

Bridge

Linea 的 Bridge 頁面，建議在 L1 預估 $4 (0.002 ETH) 以下時進行交互。

NFT 空投

之前參加測試網活動的 NFT 空投

測試網

以下為之前測試網時候的資訊

錢包

直接使用 Metamask 錢包就可以了，把測試網顯示出來即可。點擊 Show/hide test networks

然後打開

Faucet

如果要在測試網交互的話，要先到水龍頭領取測試代幣，先拿到對應 L1 的 Goerli ETH 再 Bridge 過去。以下列幾個獲取測試幣方法：

  • Alchemy: 註冊 Alchemy 帳號領取。
  • Goerli PoW Faucet: 可以用挖的。
  • Testnet Bridge: 或直接用真 ETH 買測試鏈 ETH。

另外官方也有直接領取 Linea ETH 的水龍頭：LINEA GOERLI TESTNET FAUCET

https://www.infura.io/faucet/linea

Bridge

可以使用 Hop 或 Multichain 進行 Bridge，從 Goerli 轉一些 ETH 到 Linea。參加活動會消耗蠻多 ETH，可能可以準備 0.2 ETH 持續交互。

紀念 NFT

Linea 在 Mirror 發行了紀念 NFT，使用 Optimism 主網，要準備一點 ETH。

  • Mint 費用: 0.00069 ETH
  • Gas 費用: 0.0002 ETH

所以準備大約 0.001 ETH。每週活動都會有限量的 NFT 可以 Mint，所以也可以準備多一點 ETH。
