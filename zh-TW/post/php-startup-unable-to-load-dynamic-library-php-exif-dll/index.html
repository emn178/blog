解決 PHP Startup: Unable to load dynamic library php_exif.dll

發表於2010-04-06|更新於2023-04-17|程式技術

PHP 無法啟用 exif 模組

問題

將 php.ini 的 exif 開啟如下

apache error.log 出現以下錯誤而無法開啟

原因

他必須開啟 mbstring 功能來支援且必須在 exif 之前載入，而 php.ini 預設順序卻是 exif 在前。

解決方案

將 php.ini 開啟 mbstring 並將順序置於 exif 之前。
