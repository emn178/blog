補充資訊

jscoverage 會計算 line 和 branch 兩種覆蓋率，其中 line 顧名思義就是有執行到的行數比例，而 branch 則是有執行到的條件分歧比例，例如：

其中有效行數為 5 行，會進入 if 中，所以執行行數為 4，line 的覆蓋率就是 80%；而分歧的可能情況為 2 種，執行到其中一種，所以 branch 的覆蓋率為50%。
