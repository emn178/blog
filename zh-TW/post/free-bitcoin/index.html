免費獲得比特幣

發表於2013-12-03|更新於2023-11-01|區塊鏈
|閱讀量:

提供一些免費獲得比特幣的網站，每個網站獲得的條件和時間限制都不太一樣

The Free bitcoins.com

  1. 每半小時領取
  2. 領取方式：輸入驗證碼
  3. 可在 CoinBox.Me 查訊累積金額
  4. 必須在 CoinBox.Me 累積 0.000055 BTC 以上才能匯出

Virtual Faucet

  1. 每半小時領取
  2. 領取方式：輸入驗證碼

Daily Bitcoins

  1. 每小時領取
  2. 領取方式：輸入驗證碼
  3. 必須累積 0.0001 BTC 以上才能匯出

bitcoin4.me

  1. 每小時領取
  2. 領取方式：輸入驗證碼
  3. 必須累積 0.000055 BTC 以上才能匯出

Bitcoiner

  1. 每日領取
  2. 領取方式：輸入驗證碼
  3. 必須累積 0.00013 BTC 以上才能匯出
  4. 每個小時會隨機產生可以獲得的比特幣數量，可以選擇你覺得比較多的時候領取
  5. 每日連續的領取會有額外加成，每天會增加 0.05 倍的獲得量，如果中斷的話會變回沒有加成，最多到 2 倍。

Bitcointree.net

  1. 每日領取
  2. 領取方式：輸入驗證碼
  3. 必須累積 0.0001593 BTC 以上才能匯出
  4. 每個小時會隨機產生可以獲得的比特幣數量，可以選擇你覺得比較多的時候領取

Bitvisitor

  1. 每天有不一定數量的廣告可以觀看領取
  2. 領取方式：輸入驗證碼與觀看廣告，需等 5 分鐘才能領取下一個
  3. 必須累積 0.00006 BTC 以上才能匯出

CoinAd

  1. 每天有不一定數量的廣告可以觀看領取
  2. 領取方式：輸入驗證碼與觀看廣告，每個廣告時間不同
  3. 必須累積 0.0000544 BTC 以上才能匯出
  4. 必須註冊

Ear Free Bitcoin

  1. 每天有不一定數量的廣告可以觀看領取
  2. 領取方式：輸入驗證碼與觀看廣告，需等 3 分鐘才能領取下一個
  3. 必須累積 0.000055 BTC 以上才能匯出

CoinTube.TV

  1. 每天有不一定數量的廣告可以觀看領取
  2. 領取方式：輸入驗證碼與觀看廣告，不確定是否需要觀看完畢
  3. 必須累積 0.00006 BTC 以上才能匯出

Bitcoin Addict

  1. 每日領取
  2. 領取方式：輸入驗證碼
  3. 必須累積 0.00006 BTC 以上才能匯出
  4. 有時候有機會獲得 0.01 BTC 的額外Bonus

BitHits

原本是每六小時領取，不過測試好像可以一直重復送出，不確定是不是真的可以拿到…
領取方式：直接輸入錢包位址
未說明累積多少BTC以上才能匯出
