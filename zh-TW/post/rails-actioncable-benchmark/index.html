測試結果

  1. 使用 Standalone ActionCable 比預設情況效能提升，在佔滿連線的時候也不會影響網頁和 API 使用。
  2. AnyCable 傳送訊息效能最好。
  3. 執行緒數沒有影響。
  4. ActionCable 程序數對於連線速度沒有影響，但對於傳送訊息速度可以提升。
  5. 另外測試時發現，連線數對記憶體影響不大，沒有明顯變化。
