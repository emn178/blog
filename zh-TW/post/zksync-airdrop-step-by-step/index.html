zkSync 空投交互教學

發表於2023-03-29|更新於2023-05-17|區塊鏈

本篇文章以圖文的方式來說明如何進行跨橋 (Bridge) ，以及和應用如何交互。上一篇文章 zkSync 空投策略了解策略後，我們可以開始進行交互。如果進行一次交互的策略，準備 0.02207 ETH，想要進行持續多次交易的策略，準備 0.05631 ETH。

zkSync 1.0 (Lite)

Bridge 入金

Lite 不需要加入 Network 到 Metamask，相關的操作都在以太坊主網路進行，進入 Bridge 頁面，點擊左邊 Ethereum Wallet

點擊 Top up 充值

選擇最左下角的官方 Bridge “zkSync”

這邊輸入 0.00518 ETH，右上角的手續費可以等 $5 (0.0024 ETH) 以下比較好，送出後大概要等幾分鐘。

Mint NFT

完成後上面點擊 NFT 頁籤，點擊 Mint NFT

第一次使用帳號要進行啟用，點擊按鈕啟用

還要再進行一次簽章

可以自己上傳檔案到 IPFS 取得 Content Address，或者只是要互動的話，可以利用 STORJ 來生成，拖曳檔案或點擊 click to browse 來上傳圖片

他會產生一個 Hash

把那個 Hash 貼到 Content Address 送出

完成

由於提領 NFT 到 L1 要花的手續費蠻高的預估要 0.00857 ETH，我就沒有操作了。

ZigZag

ZigZag 似乎是 Lite 上唯一的應用，進入網頁後左邊點擊 SELL 頁籤，先連接錢包。最小交易額為下方的 Network fee，例如現在是 0.0000875 ETH，可以輸入 0.0000876

之後每隔一段時間來交易一次，最後一次可以把所有 USDC 買回 ETH，然後在全部 Bridge 回 L1。

Bridge 出金

之後不再交互想要把資產 Bridge 回 L1 的話，先回到 Bridge 首頁，點擊 Transfer

點擊 Transfer to Ethereum

點 Max 可全部轉出，下面的 Fee 大約 0.00308 ETH ($5.3，現在約 $6) 勉強可接受

zkSync 2.0 (Era)

Bridge 入金

這邊以 Portal 為例，進入網頁後輸入 0.04523 然後點擊 Deposit，建議可以等到手續費 $6 (0.003 ETH) 以下。

送出後一樣要等幾分鐘

Mute.Switch

Bridge 完成後，接著進入 Mute.Switch，連結錢包的過程會自動跳出加入 zkSync Era Maiinnet 的畫面，點擊批准

然後切換網路，之後的操作都在這條鏈上進行

中間點擊 Select Token，上面選擇 ETH 下面選擇 USDC，輸入等等想放入池子的量，例如 0.0001

接著到 Pool 頁面，點擊 USDC / ETH 右邊的 Manage

點擊 USDC 上面的 MAX

點 Use default 之後不用重複授權，或點「最大值」只授權這次要轉入的，我這邊選擇「最大值」。授權完後會立刻再跳一筆入金的交易

如果之後要出金的話，在 Pool 頁面看到自己存入的部分，點擊右邊的 Manage 可以進行取出

上面切到 Withdraw，點擊 MAX 可全部取出，先點擊 Approve 授權，授權完後會再跳一筆出金的交易

SyncSwap

SyncSwap 操作和 Mute.Switch 一樣，中間選擇 ETH 換 USDC

接著到 Pool 頁面，選擇 USDC / ETH

左邊切到 Deposit，打開 Add tokens in balanced proportion，點擊 USDC 旁的 MAX，再點擊 Unlock USDC。交易完成後 Deposit 按鈕變成可以按的，最後在點擊 Deposit 才能完成入金。

之後如果要出金的話，在左邊的 Withdraw 進行，中間 Single 可選擇全部變成 USDC 或 ETH 提出，或是切到 Balanced 照比例提出 USDC 和 ETH。如果之後是全部 Bridge 回 L1，可以選全部變成 ETH。這邊要先 Permit LP Token，再按 Withdraw

(04/07 更新)
SyncSwap 04/03 開始有 Galxe活動可以參加，到 04/20，第三點條件是交易兩次，並且 Deposit 生成一個 Position，所以上面的步驟做完之後只要在交易一次就可以了。SyncSwap 官方已經公告要發幣了，可以參加看看。

Mint Square

進入 Mint Square 點擊 Mint

隨便上傳圖片和輸入一個名字，再點擊最下面的 Mint，會先連結錢包

Mint 完成，點擊 Go to Profile

可以看到自己的 NFT

點進自己的 NFT 可以看到這畫面，下面已經有很多人出價要買…?

點擊 Sell，要先 Unlock NFT

完成後可以隨便掛一個價格上架出售

結果真的有人買，應該是官方機器人在造勢

zkSync Name Service

進入 zkSync Name Service，點擊右上 LAUNCH APP

右上點擊 Connect Wallet 連接錢包，中間搜索輸入想註冊的域名，輸入 5 個字元以上最便宜，例如: mytest，請改成你自己的

按 Search 送出後可以看到能不能註冊，右邊顯示 Available 表示可以，點擊左邊白色的 mytest.zk 進入下一步驟

可以看到現在註冊要 0.0029 ETH，點最下面 Request to Register with ETH

完成之後點上面 My Profile 可以看到註冊的名稱，Reverse Record 選擇剛剛買的名稱按 Save

04/08 前還可以進行 Galxe 活動，不過完成任務可以領到的 NFT 是在 Polygon 鏈上，要有 MATIC 才能領取，手續費 0.03 MATIC 左右。進入網頁會先連接錢包，然後點擊右邊的 Connect Twitter

第一次使用 Galxe 會先綁定 Twitter 帳號，點擊 Tweet 發文

然後左下點擊 Profile，右邊找到剛發的文

右下角的 Share 點開

選 Copy link to Tweet 複製到剪貼簿

然後在 Step 2 貼上，按 Verify

現在 Connect Twitter 完成之後，部分需要 Twitter 帳號的任務解鎖，剛發的 Tweet 也可以刪了。先點擊右邊的 Follow, Quote 和 Like 按鈕去完成就可以了，Quote 需要 Tag 三個朋友。由於 Verify 有時候會延遲，最後全部做完再一起 Verify。接著點擊 Connect Discord，授權後點 Join Now 加入

在最新的訊息下面點擊星星圖示

同意規則

回到 Galxe 點擊 Verify Email，輸入 Email 之後點擊 Send a code，收信把認證碼輸入

綁定 Email 之後他會解鎖，出現 Go 的按鈕，點擊 Go 去外部頁面

點擊 Continue to Access

最後回來進行 Verify，然後點擊左邊的 Switch to POLYGON 按鈕，會新增 Polygon 網路並切過去，然後就可以按 Claim 領取 (錢包要有 MATIC)。

Bridge 出金

之後可以每隔一段時間來交互一次，不再交互想要Bridge 回到 L1 的話，可以到 Bridge 頁面，點擊 MAX 全部提出，點擊 Withdraw 執行。手續費可以在 $1 左右進行，例如現在 0.00063 ETH，

Zealy

進入 zkSync 的任務進行登入，我選擇 Metamask 登入，之後有獎勵的話可以用這個錢包接收

接著可以看到任務列表，必須一關一關完成才會開放更多，
