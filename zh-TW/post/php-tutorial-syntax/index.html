註解的用法

註解的部份不會執行，是給人看的，PHP中有三種註解方式：

  1. // 單行註解：兩個斜線後面的部份將不會處理。
  2. # 單行註解：和1.一樣功能，是shell-style的寫法。
  3. /* ... */ 多行註解：註解中間的部份將不會處理。

註解的技巧

我們可以利用 //* ... //*/ 的寫法來作為區塊註解的開關
